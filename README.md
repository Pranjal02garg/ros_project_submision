# Autonomous Indoor Cleaning Robot using ROS2, Gazebo, SLAM, Nav2, and PID Control

This project simulates an autonomous indoor cleaning robot in a custom apartment environment using ROS2 Humble, Gazebo Classic, RViz, SLAM Toolbox, and Nav2.

## Features

- Four-wheeled differential drive mobile robot
- URDF robot model
- Custom apartment world in Gazebo
- 2D LiDAR sensor
- RViz visualization
- Keyboard teleoperation
- Custom PID movement controller
- SLAM mapping using SLAM Toolbox
- Saved occupancy grid map
- Autonomous navigation using Nav2
